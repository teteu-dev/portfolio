MINHAS ESPECIALIDADES.

HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação de hipertexto que serve para criar páginas web. É o componente básico da web e permite a inserção de conteúdo como imagens, vídeos e textos.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo que descreve como um documento é apresentado. O CSS é usado para estilizar elementos escritos em HTML ou XML.

JavaScript

JavaScript é uma linguagem de programação interpretada, baseada em objetos e usada para criar scripts em páginas web. É também conhecida como JS.
